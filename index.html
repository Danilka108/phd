РЕФЕРАТ

Отчет с., источн., рис., табл.


автоматизированное развертывание, изолированная среда с ограниченными ресурсами, централизованное управление кластером сервисов


Разрабатываемая информационная система представляет собой подсистему администрирования для развертывания и управления программными компонентами системы контроля и управления доступом (СКУД) для группы лифтов в условиях ограниченного доступа к сети и ограниченных ресурсов. Система обеспечивает полный цикл управления жизненным циклом сервисов — от установки и конфигурации до мониторинга и удаления — без участия администратора и подключения к интернету.

Отличительными особенностями системы являются: модульная архитектура, позволяющая разворачивать СКУД на множестве устройств; использование автономного установочного пакета для развертывания системы без доступа к интернету; механизм согласования топологии кластера, автоматически приводящий фактическое состояние системы к заданной топологии; использование утилиты командной строки для управления системой.

Практическая значимость работы заключается в снижении эксплуатационных затрат, устранении ошибок ручной настройки, повышении надёжности и воспроизводимости при внедрении и сопровождении СКУД для группы лифтов.
